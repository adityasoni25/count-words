# Count Words Assignment

## ✅ Problem Statement
A Java program that processes a list of words to:
- Count the number that start with **M** or **m**
- Return all words longer than 5 characters

## 🏗️ Tech Stack
- Java 17
- Spring Boot
- Maven
- JUnit 5
- Docker

## 🚀 How to Run

### Run Locally
```bash
mvn spring-boot:run
